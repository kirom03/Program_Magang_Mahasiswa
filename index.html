Tentang

Apa itu program magang UBHI

Program magang di Universitas Bhineka PGRI Tulungagung adalah kesempatan bagi mahasiswa untuk belajar langsung di dunia kerja. Melalui program ini, mahasiswa bisa mengembangkan keterampilan, menambah pengalaman, dan memahami cara kerja di berbagai instansi atau perusahaan. Magang ini terbuka untuk semua jurusan dan menjadi bagian dari dukungan kami terhadap :

Pertanyaan yang Sering Diajukan

Pertanyaan Umum Seputar Program Magang UBHI

Anda dapat mendaftar melalui website resmi UBHI. Pilih program magang yang sesuai dengan minat Anda, lalu isi formulir pendaftaran yang tersedia. Pastikan semua data yang Anda masukkan sudah benar sebelum mengirimkan formulir.

Persyaratan umum meliputi: status sebagai mahasiswa aktif, CV terbaru, surat pengantar dari universitas, dan portofolio (jika diperlukan). Persyaratan tambahan dapat berbeda tergantung pada program magang yang Anda pilih.

Tidak, program magang UBHI tidak memungut biaya pendaftaran. Namun, Anda perlu menanggung biaya pribadi seperti transportasi dan akomodasi selama magang.

Ya, setiap peserta yang menyelesaikan program magang UBHI akan mendapatkan sertifikat resmi sebagai bukti partisipasi dan pencapaian.
